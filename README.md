## Resumo: Atividade de Lógica
Repositório dedicado à resolução de problemas de lógica de programação utilizando representação visual e textual.

## Tecnologias
Flowgorithm: Criação e execução de fluxogramas (.fprg).

VisuAlg: Implementação de algoritmos em pseudocódigo/Portugal (.alg).

## Organização
Fluxogramas: Lógica visual e estrutural.
 Como testar
--
Abrir arquivos .fprg no Flowgorithm e clicar em "Executar".

Abrir arquivos .alg no VisuAlg e pressionar F9.
